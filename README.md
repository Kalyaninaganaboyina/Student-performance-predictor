  # 🎓 Student Performance Predictor

## 📌 Project Overview
This project predicts student math scores using Machine Learning techniques based on features like reading score, writing score, lunch type, and test preparation.

## 🧠 Problem Type
Regression Problem (predicting continuous values)

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Models Used
- Linear Regression
- Decision Tree Regressor

## 📈 Evaluation Metrics
- Mean Absolute Error (MAE)
- R2 Score

## 🚀 Features
- Data preprocessing
- Feature encoding
- Model training & testing
- Model comparison
- Visualization

## ▶️ How to Run
1. Install dependencies:
   pip install pandas matplotlib scikit-learn

2. Run:
   python Student_performance.py
