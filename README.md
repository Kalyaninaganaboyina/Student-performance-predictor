# Student Performance Predictor

## 📌 Description
This project predicts student math scores using Machine Learning.

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Models Used
- Linear Regression
- Decision Tree Regressor

## 📈 Evaluation Metrics
- Mean Absolute Error (MAE)
- R2 Score

1. Install libraries
